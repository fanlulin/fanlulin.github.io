VUE+FLASK+YOLOV5物品识别平台

Created At : 2022-07-02 09:44
Views 👀 :
  1. 前端页面主要是环境的搭建
  2. 后端项目主要是环境的配置

代码下载：git clone https://github.com/liuxiaoxiao666/Yolov5-Flask-VUE.git
包含前端页面和后端页面。

前端页面主要是环境的搭建

进入前端文件夹：cd front-end/
安装cnpm淘宝镜像，npm有一些问题，安装不上去

在使用cnpm install就可以下载依赖了
最后使用npm run serve启动前端项目。

后端项目主要是环境的配置

  1. pip install -r requirements.txt安装到自己的虚拟环境中
  2. pip install flask等等一些库
  3. from … import …会出现找不到自己的库的情况
    右键项目文件夹,make directory后选择source root，就可以找到了。
  4. slow_conv2d_cpu” not implemented for ‘Half’
    将model.half()，img = img.half()修改成model.float()，img = img.float()就可以了。
  5. Upsample’ object has no attribute ‘recompute_scale_factor’
    参考链接：https://blog.csdn.net/qq_40280673/article/details/125095353
    将upsampling.py的第151删掉最后一个参数。
